# n8n Automation Portfolio

A collection of AI-powered business automation workflows built with **n8n**, LLMs, cloud databases, and third-party integrations. These projects are designed to reduce manual work, streamline business operations, and showcase production-oriented automation architecture using AI agents and modern cloud services.

## Projects

### AI Recruitment Intelligence Platform

An AI-powered recruitment system that automates candidate intake, resume analysis, and recruiter-ready evaluations.

#### Features

* Candidate application portal with resume upload
* AI-powered resume parsing
* RAG-powered hiring knowledge base
* Automated candidate screening and evaluation
* Weighted candidate scoring (0–100)
* Recruiter summary generation
* Automated email notifications
* Candidate pipeline management
* Error monitoring and alerting
* Supabase database and storage integration

#### Tech Stack

* n8n
* Supabase
* Groq API
* pgvector
* Gmail API
* HTML
* CSS
* JavaScript

### AI Customer Support Automation

An AI-powered customer support workflow that automates email handling, multilingual responses, ticket management, and human approval for critical issues.

#### Features

* Gmail inbox monitoring

* Multilingual response generation
* Customer issue summarization
* Priority classification (High, Medium, Low)
* Google Docs knowledge base integration
* Airtable ticket creation and tracking
* Slack notifications for high-priority tickets
* Human approval workflow
* Automated customer replies
* Structured JSON output for downstream automation

#### Tech Stack

* n8n
* Google Gemini
* Gmail API
* Airtable
* Slack API
* Google Docs API
